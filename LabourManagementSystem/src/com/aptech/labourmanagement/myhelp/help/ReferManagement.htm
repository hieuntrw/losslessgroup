Refer Management

Functions:

- Add new Refer:

       + Click “Add” button:

       + Input information into the fields then click “Save” button to save:

+ If valid information, this message dialog will appear

+ Otherwise the warning dialog will appear

 

-  Edit Refer:

       + Select Refer on “Refer list” table:

       + Click “Edit” button then rewrite the information that you want to change:

       + Click “Save” button. If create successful message dialog appear

+ Or “Cancel” button or cancel

-  Delete Refer:

       + Select Refer on “Refer list” table:

       + Click “Delete” button: Confirm dialog appear

+ Choose “Yes” to delete this Refer

+ Or “No” to cancel
