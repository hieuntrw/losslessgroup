Labor Management

 

 

Functions:

-  Add new Labor:

      + Click “Add” button:

+ Input information into the fields then click the “Save” button:

      + If valid information, this message will appear

+ Otherwise the message dialog error will appear

 

- Edit Labor:

+ Select Labor on “Labor list” table:

      + Click “Edit” button then rewrite information that you want change:

      + Click “Save” button

+ Or “Cancel” button to save information or cancel

- Delete Labor:

+ Select Labor on “Labor list” table:

      + Click “Delete” button:

      + Choose “Yes” to delete Labor

+ Or select “No” to cancel
